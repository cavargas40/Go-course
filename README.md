## Commands
Check Version
    - `$ go version` 

Run Project
    - `$ go run <filename>.go`

Documentation
    - `$ go doc <library>` ex: time

Format document
    - `$ go fmt <filename>.go`

Build Program - generates an executable file
    - `$ go build <filename>.go`

Install go package
    - `$ go get -u <packagename>`
    - `$ go get -u github.com/gorilla/mux`
